# Python

A large, curated collection of Python scripts and Jupyter notebooks covering fundamentals, advanced algorithms, GUI programming, automation, web scraping, data science libraries, competitive programming, and mini-projects. Ideal as a learning reference or a personal utility toolkit.

## Features

- **150+ standalone scripts** — each file is self-contained and runnable
- **Algorithm implementations** — sorting, searching, graph algorithms, dynamic programming, cryptography
- **Data structures** — AVL trees, B+ trees, Fibonacci heaps, segment trees, skip lists, tries, treaps, red-black trees
- **GUI applications** — Tkinter-based calculator, calendar, music player, paint app, text editor, games
- **Automation** — WhatsApp bots, email automation, Google Forms filler, birthday wisher, PDF tools
- **Web scraping** — IMDB, Reddit, Hacker News, LinkedIn, Moneycontrol, Medium, Flipkart (Selenium)
- **Mini games** — Snake, Blackjack, Rock-Paper-Scissors, Hangman, Sudoku, Tic Tac Toe with AI
- **Data science notebooks** — NumPy, Pandas, Matplotlib, Seaborn, Python basics, competitive programming

## Getting Started

### Prerequisites

```bash
pip install pygame selenium requests beautifulsoup4 pillow pyautogui pyttsx3 fpdf mpi4py
```

> Individual scripts may have additional dependencies. Check the import block at the top of each file.

### Run a Script

```bash
python "Snake Game.py"
python "Dijkstra's Shortest Path.py"
python "Chatbot-GUI-Tkinter.py"
```

### Open Notebooks

```bash
jupyter notebook "Python Basic - 1.ipynb"
jupyter notebook "Numpy.ipynb"
jupyter notebook "Python CP - 1.ipynb"
```
